Como Este Manual Funciona

Bem-vindo ao nosso manual dinâmico! Esta plataforma foi construída como uma Aplicação de Página Única (SPA), o que significa que o conteúdo é carregado de forma rápida e fluida, sem a necessidade de recarregar a página inteira a cada clique.

Os Componentes Principais

A mágica acontece através da interação de três arquivos principais:

  • index.html (A Estrutura): Este arquivo que você está vendo agora. Ele funciona como o "esqueleto" visual, definindo a área do menu lateral e este espaço principal para o conteúdo.
  • routes/routes.json (O Mapa): Um arquivo de configuração que define toda a estrutura do menu de navegação que você vê à esquerda. Ele diz ao sistema quais tópicos existem e onde encontrar o arquivo de conteúdo para cada um.
  • assets/js/main.js (O Motor): O "cérebro" da aplicação. Este script lê o mapa (routes.json), constrói o menu de navegação dinamicamente e, quando você clica em um link, ele busca o conteúdo correspondente e o exibe aqui.

Como Navegar

Agora que você sabe como funciona, sinta-se à vontade para explorar! Use o menu lateral para navegar pelos diferentes tópicos da documentação.
